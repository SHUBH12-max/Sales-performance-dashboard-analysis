# Sales Performance Dashboard Analysis

## Project Overview

This Power BI project analyzes sales performance data to provide insights into revenue, customers, products, and regional performance through interactive dashboards.

## Tools Used

* Power BI
* DAX
* Power Query

## Key KPIs

* Total Sales
* Total Orders
* Total Customers

## Dashboard Features

* Sales Trend Analysis
* Customer Insights
* Regional Sales Comparison
* Product Performance Analysis
